## For loop

El script multiplicacion.js imprime la tabla del 1 al ser ejecutado. Modifica el script para que imprima la tabla que el usuario indique usando argumentos de linea de comandos.

Ejemplo:
- `node multiplicacion.js 5`
- `5 x 1 = 5`
- `5 x 2 = 10`
- `5 x 3 = 15`
- `5 x 4 = 20`
- `5 x 5 = 25`
- `5 x 6 = 30`
- `5 x 7 = 35`
- `5 x 8 = 40`
- `5 x 9 = 45`
- `5 x 10 = 50`
